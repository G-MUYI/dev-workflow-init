# 🚀 5 分钟快速上手指南

> 适合第一次使用的小白用户

## 这是什么？

这是一个 AI 助手的"技能包"，可以帮你：
- 通过对话的方式梳理项目需求
- 自动生成项目规范文档
- 推荐适合你项目的工具
- 规划开发流程

**就像和 AI 聊天一样简单！**

---

## 第一步：安装到你的项目

### Windows 用户

1. 下载这个项目到电脑（比如放在 `D:\skills\dev-workflow-init`）
2. 打开你想要初始化的项目文件夹（比如 `D:\my-project`）
3. 在项目文件夹里，按住 `Shift` 键，右键点击空白处，选择"在此处打开 PowerShell 窗口"或"在此处打开命令提示符"
4. 输入以下命令（记得改成你的实际路径）：

```cmd
D:\skills\dev-workflow-init\install-unified.bat
```

5. 按回车，等待安装完成

### Mac/Linux 用户

1. 下载这个项目到电脑（比如放在 `~/skills/dev-workflow-init`）
2. 打开终端（Terminal）
3. 进入你的项目文件夹：

```bash
cd ~/my-project
```

4. 运行安装命令：

```bash
bash ~/skills/dev-workflow-init/install-unified.sh
```

5. 等待安装完成

---

## 第二步：启动 AI 对话

安装完成后：

1. 用你的 AI 编程工具打开项目（比如 Claude Code、Cursor、VSCode 等）
2. 在 AI 对话框中输入：

```
启动
```

或者

```
start
```

3. AI 会开始问你问题，像这样：

```
你好！我会通过 8 个问题帮你梳理项目需求...

现在开始第一个问题：

1️⃣ 你想实现的核心效果是什么？请用 1-2 句话描述。
```

---

## 第三步：回答问题

AI 会问你 8 个问题，比如：

- 你的项目要做什么？
- 给谁用的？
- 用什么技术？
- 什么时候完成？
- ...

**你只需要像聊天一样回答就行！**

例如：
```
问：你想实现的核心效果是什么？
答：我想做一个在线记账本，可以记录每天的收支

问：主要面向哪些用户？
答：个人用户，主要是想管理自己日常开销的人
```

---

## 第四步：确认并生成

回答完 8 个问题后，AI 会给你看一个摘要：

```
📋 需求摘要确认

1. 核心目标：在线记账本，记录每天收支
2. 目标用户：个人用户
3. 成功标准：...
...

请确认以上信息是否正确
```

确认后，AI 会自动：
- 生成 `AGENTS.md` 文件（项目规范）
- 推荐适合的工具和插件
- 规划开发步骤

## 常见问题

### ❓ 我说"启动"后 AI 没反应？

**可能原因：**
- 安装脚本没有正确运行
- AI 工具没有识别到 skill

**解决方法：**
1. 检查项目文件夹里是否有 `.ai` 文件夹
2. 重新运行安装脚本
3. 重启你的 AI 编程工具
4. 或者明确说："请使用 dev-workflow-init skill"

### ❓ 我不知道怎么回答某个问题？

**没关系！** 你可以：
- 说"跳过"，AI 会继续下一个问题
- 说"不确定"，AI 会给你一些建议
- 随便说点什么，AI 会根据你的回答调整

### ❓ 我可以中途暂停吗？

**可以！** 说"暂停"或"稍后继续"，AI 会保存进度。下次说"继续"就能恢复。

### ❓ 我想重新开始怎么办？

说"重来"或"重新开始"，AI 会清空之前的回答，从第一个问题重新开始。
