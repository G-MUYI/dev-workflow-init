# 使用示例

## 场景 1：直接启动

**用户输入：**
```
启动
```

**AI 应该回复：**
```
你好！我会通过 8 个问题帮你梳理项目需求，然后生成 AGENTS.md 规则文件、推荐适配的 skills 和工具、编排开发工作流。

现在开始第一个问题：

你想实现的核心效果是什么？请用 1-2 句话描述。
```

---

## 场景 2：询问式触发

**用户输入：**
```
我想开始一个新项目，不知道从哪里开始
```

**AI 应该回复：**
```
看起来你想初始化一个新项目的开发工作流。我可以通过访谈帮你生成 AGENTS.md 和推荐工具，要开始吗？（回复「是」或「启动」）
```

**用户确认后：**
```
你好！我会通过 8 个问题帮你梳理项目需求...
```

## 常见问题

### Q: AI 没有识别我的"启动"指令怎么办？

A: 尝试以下方法：
1. 明确说："请使用 dev-workflow-init skill"
2. 或者说："帮我初始化项目开发工作流"
3. 检查 skill 是否正确安装在 `.claude/skills/` 目录下

### Q: 可以中途暂停访谈吗？

A: 可以。说"暂停"或"稍后继续"，AI 会保存当前进度。下次说"继续"即可恢复。

### Q: 如果想重新开始怎么办？

A: 说"重来"或"重新开始"，AI 会清空已收集的信息，从第一个问题重新开始。
